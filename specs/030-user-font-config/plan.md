# Implementation Plan: User Font Configuration

**Branch**: `030-user-font-config` | **Date**: 2025-12-26 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/030-user-font-config/spec.md`

## Summary

Enable users to declaratively configure fonts in their user config with support for:

1. Public fonts from nixpkgs via `fonts.packages`
1. Private fonts from git repositories via `fonts.repositories` (using deploy key)
1. Default desktop font configuration for GNOME (macOS has limited support)

## Technical Context

**Language/Version**: Nix (flakes enabled, 2.19+)\
**Primary Dependencies**: Home Manager, agenix (secrets), nix-darwin, NixOS\
**Storage**: User font directories (~/.local/share/fonts/, ~/Library/Fonts/)\
**Testing**: `nix flake check`, manual activation testing\
**Target Platform**: macOS (darwin), Linux (NixOS, GNOME family)
**Project Type**: Nix configuration repository\
**Performance Goals**: Font installation < 60s, private repo clone < 30s\
**Constraints**: Must work without deploy key (graceful skip)\
**Scale/Scope**: Single-user, multiple font repositories

**Structure Decision**: Settings modules for cross-platform (shared) and platform-specific (darwin, gnome) font handling. Core logic in shared, desktop-specific configuration in family/platform modules.

## Module Responsibilities

### system/shared/settings/fonts.nix (Core)

- Read `config.user.fonts` configuration
- Install packages from `fonts.packages` via `home.packages`
- Deploy SSH key for private repos (using secrets helper)
- Clone/update repositories from `fonts.repositories`
- Copy font files to user font directory
- Refresh font cache (Linux)

### system/shared/family/gnome/settings/fonts.nix (GNOME)

- Read `config.user.fonts.default`
- Configure GNOME monospace font via dconf
- Map package name to font family name

### system/darwin/settings/fonts.nix (Darwin)

- Currently a placeholder (no-op)
- macOS lacks centralized monospace font setting
- Individual apps configure their own fonts

### user/shared/lib/home-manager.nix (Schema)

- Add `fonts` option with submodule type
- Fields: `default`, `packages`, `repositories`
