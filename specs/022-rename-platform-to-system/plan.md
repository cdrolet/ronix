## Summary

Rename the `platform/` directory to `system/` to align directory naming with constitution terminology. The directory name currently conflicts with the `platform` field in user definitions (which refers to OS platform like "darwin"/"nixos"). This change updates directory paths and documentation while preserving git history and keeping the `platform` field name unchanged.
