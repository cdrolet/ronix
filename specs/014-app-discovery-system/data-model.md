## Core Data Structures

### Application Specification

```nix
# Type: ApplicationSpec
# Description: User-facing specification of applications to install
# Location: User configs (user/*/default.nix) or profiles (system/*/profiles/*/default.nix)

{
  applications = [
    # Simple name (auto-discovered)
    "git"           # String
    "zsh"
    "helix"
    
    # Partial path (disambiguation)
    "darwin/aerospace"        # String with "/"
    "shared/app/dev/git"
    
    # Full path (explicit)
    "system/shared/app/editor/helix"  # String starting with base path
  ];
}

# Type Signature
applications :: List String
```

### App Resolution Result

```nix
# Type: AppResolution
# Description: Internal result of resolving an app name to a path
# Used by: discovery.resolveApplications

{
  # Input app name
  appName = "git";              # String - original input
  
  # Resolution result
  resolvedPath = /nix/store/.../system/shared/app/dev/git.nix;  # Path
  
  # Search metadata
  searchPaths = [
    "system/darwin/app/git.nix"          # String - checked first
    "system/shared/app/**/git.nix"       # String - matched here
  ];
  
  # Match metadata
  matchType = "recursive-search";  # "recursive-search" | "partial-path" | "exact-path"
  matchedAt = "system/shared/app/dev/git.nix";  # String - relative path
  
  # Timing (optional, for profiling)
  searchTimeMs = 12;  # Int
}

# Type Signature
AppResolution :: {
  appName :: String,
  resolvedPath :: Path,
  searchPaths :: List String,
  matchType :: String,
  matchedAt :: String,
  searchTimeMs :: Maybe Int
}
```

### Caller Context

```nix
# Type: CallerContext
# Description: Metadata about where resolveApplications was called from
# Used by: discovery.detectContext

{
  # Caller identification
  callerPath = ./user/cdrokar/default.nix;  # Path - absolute path to caller file
  callerType = "user-config";  # "user-config" | "darwin-profile" | "nixos-profile" | "unknown"
  
  # Platform detection
  platform = "darwin";  # "darwin" | "nixos" | "shared" | null
  
  # Search context
  basePath = ./.;  # Path - repository root
  relativePath = "user/cdrokar";  # String - relative from root
}

# Type Signature
CallerContext :: {
  callerPath :: Path,
  callerType :: String,
  platform :: Maybe String,
  basePath :: Path,
  relativePath :: String
}

# Caller Types
CallerType = 
  | "user-config"      # Called from user/*/default.nix
  | "darwin-profile"   # Called from system/darwin/profiles/*/default.nix
  | "nixos-profile"    # Called from system/nixos/profiles/*/default.nix
  | "unknown"          # Other location
```

### Search Path Configuration

```nix
# Type: SearchPathConfig
# Description: Ordered list of paths to search for apps
# Generated by: buildSearchPaths based on CallerContext

{
  # Ordered search paths (priority: first to last)
  paths = [
    {
      basePath = "system/darwin/app";    # String - directory to search
      recursive = true;                   # Bool - search subdirectories
      priority = 1;                       # Int - lower = higher priority
      reason = "Platform-specific apps";  # String - why this path
    }
    {
      basePath = "system/shared/app";
      recursive = true;
      priority = 2;
      reason = "Shared cross-platform apps";
    }
  ];
  
  # Context metadata
  context = <CallerContext>;  # CallerContext - original context
}

# Type Signature
SearchPathConfig :: {
  paths :: List SearchPath,
  context :: CallerContext
}

SearchPath :: {
  basePath :: String,
  recursive :: Bool,
  priority :: Int,
  reason :: String
}
```

```nix
# Description: Structured error when app cannot be found

{
  # Error identification
  errorType = "app-not-found";  # "app-not-found" | "ambiguous-match" | "invalid-path"
  
  # Error details
  appName = "aerospacee";  # String - requested app name
  searchedPaths = [
    "system/darwin/app/aerospacee.nix"
    "system/shared/app/**/aerospacee.nix"

  # Suggestions
  suggestions = [
    { name = "aerospace"; path = "system/darwin/app/aerospace.nix"; similarity = 0.95; }
    { name = "airflow"; path = "system/shared/app/dev/airflow.nix"; similarity = 0.72; }
  ];
  
  # Context
  callerLocation = "user/cdrokar/default.nix:15";  # String
  
  # Error message
  message = ''
    error: Application 'aerospacee' not found
    
    Searched locations:
      - system/darwin/app/aerospacee.nix (not found)
      - system/shared/app/**/aerospacee.nix (not found)
    
    Did you mean one of these?
      - aerospace (system/darwin/app/aerospace.nix)
      - airflow (system/shared/app/dev/airflow.nix)
    
    Called from: user/cdrokar/default.nix:15
  '';
}

# Type Signature
AppDiscoveryError :: {
  errorType :: String,
  appName :: String,
  searchedPaths :: List String,
  suggestions :: List Suggestion,
  callerLocation :: String,
  message :: String

  name :: String,
```

```nix
# Main entry point for app discovery

resolveApplications :: {
  apps :: List String,           # App names to resolve
  callerPath :: Path,            # Path of calling file
```

## Search Algorithm

### Recursive App Search

```
function findApp(appName, searchPath, basePath):
  fullPath = basePath + "/" + searchPath.basePath
  
  if not exists(fullPath):
    return null
  
  if searchPath.recursive:
    # Recursive search
    for each file in recursiveListDir(fullPath):
      if file.name == appName + ".nix":
        return file.path
      if file.name == appName and file.isDirectory:
        # Check for app/default.nix pattern
        if exists(file.path + "/default.nix"):
          return file.path + "/default.nix"
    return null
  else:
    # Non-recursive search
    directPath = fullPath + "/" + appName + ".nix"
    if exists(directPath):
      return directPath
    
    dirPath = fullPath + "/" + appName + "/default.nix"
    if exists(dirPath):
      return dirPath
    
    return null
```

### Path Matching Logic

```
function matchPath(appName, searchPath):
  if appName contains "/":
    # Partial or full path specified
    if appName startsWith "system/":
      # Full path from root
      return exactMatch(appName)
    else:
      # Partial path - find first match containing this path
      return partialMatch(appName, searchPath)
  else:
    # Simple name - recursive search
    return findApp(appName, searchPath)
```

## Priority Rules

### Search Priority by Context

**From User Config** (`user/*/default.nix`):

```
Priority 1: system/shared/app/**/*.nix     (shared, recursive)
Priority 2: system/darwin/app/**/*.nix     (if darwin exists, recursive)
Priority 3: system/nixos/app/**/*.nix      (if nixos exists, recursive)
```

**From Darwin Profile** (`system/darwin/profiles/*/default.nix`):

```
Priority 1: system/darwin/app/**/*.nix     (platform-specific, recursive)
Priority 2: system/shared/app/**/*.nix     (shared, recursive)
```

**From NixOS Profile** (`system/nixos/profiles/*/default.nix`):

```
Priority 1: system/nixos/app/**/*.nix      (platform-specific, recursive)
Priority 2: system/shared/app/**/*.nix     (shared, recursive)
```

### Disambiguation Rules

When multiple matches exist:

1. **Exact path wins**: `"system/shared/app/dev/git"` matches exactly
1. **Closer platform wins**: `darwin/app/aerospace` over `shared/app/aerospace`
1. **First in priority order wins**: If multiple matches at same priority, use first found
1. **Error if ambiguous**: When path is ambiguous, show all matches and require disambiguation

## State Transitions

### App Resolution State Machine

```
[App Name] 
    ↓
[Detect Caller Context]
    ↓
[Build Search Paths]
    ↓
[Parse App Name]
    ├─ Contains "/" → [Partial/Full Path Match]
    └─ Simple name → [Recursive Search]
         ↓
[Search Priority 1] ─→ Match? → [Return Path]
         ↓ No
[Search Priority 2] ─→ Match? → [Return Path]
         ↓ No
[Search Priority N] ─→ Match? → [Return Path]
         ↓ No
[Generate Suggestions]
         ↓
[Throw Error with Suggestions]
```

## Example Data Flows

### Flow 1: User Config with Simple Names

```nix
# Input
applications = [ "git" "zsh" ];
callerPath = ./user/cdrokar/default.nix;

# Step 1: Detect Context
{
  callerType = "user-config";
  platform = null;
  basePath = ./.;
}

# Step 2: Build Search Paths
[
  { basePath = "system/shared/app"; priority = 1; recursive = true; }
  { basePath = "system/darwin/app"; priority = 2; recursive = true; }
]

# Step 3: Resolve "git"
Search: system/shared/app/**/*.nix
Match: system/shared/app/dev/git.nix ✓

# Step 4: Resolve "zsh"
Search: system/shared/app/**/*.nix
Match: system/shared/app/shell/zsh.nix ✓

# Output
[
  /nix/store/.../system/shared/app/dev/git.nix
  /nix/store/.../system/shared/app/shell/zsh.nix
]
```

### Flow 2: Darwin Profile with Platform-Specific App

```nix

applications = [ "aerospace" ];

{
  callerType = "darwin-profile";
  platform = "darwin";
  basePath = ./.;

# Step 2: Build Search Paths
```

## Performance Considerations

### Caching Strategy

```nix
# Cache at build time
appCache = {
  "git" = /nix/store/.../system/shared/app/dev/git.nix;
  "zsh" = /nix/store/.../system/shared/app/shell/zsh.nix;
  # ... cached for entire build
};

# Cache key: (appName, callerContext)
cacheKey = "${appName}-${callerContext.callerType}";
```

### Search Optimization

- Use `builtins.readDir` (fast, built-in)
- Early exit on first match
- Skip non-existent directories
- Lazy evaluation for suggestions (only on error)

### Complexity Analysis

```
n = number of apps in list
m = number of files in search paths
d = max directory depth

Time Complexity:
- Best case: O(n) - all apps in first search path
- Worst case: O(n * m * d) - all apps searched recursively
- Average case: O(n * log(m)) - most apps found quickly

Space Complexity: O(n) - store resolved paths
```

## Constraints & Invariants

### Constraints

1. **App names must be unique** within same search priority level
1. **Paths must be relative** to repository root
1. **App files must be .nix** or directories with default.nix
1. **Search paths must exist** (skipped if not)
1. **Caller path must be absolute** (for context detection)

### Invariants

1. **Resolution deterministic**: Same input → same output (no randomness)
1. **Priority respected**: Higher priority paths always checked first
1. **No duplicate imports**: Same app not imported twice
1. **Error on missing**: Unknown apps cause build failure (fail-fast)
1. **Backward compatible**: Manual imports unaffected

## Validation Rules

### App Name Validation

```nix
# Valid app names
"git"                     ✓ Simple name
"darwin/aerospace"        ✓ Partial path
"system/shared/app/dev/git" ✓ Full path
"git.nix"                 ✓ With extension

# Invalid app names
""                        ✗ Empty string
"../git"                  ✗ Parent directory reference
"/absolute/path"          ✗ Absolute path
"app with spaces"         ✗ Contains spaces
```

### Search Path Validation

```nix
# Valid search paths
"system/shared/app"       ✓ Relative directory
"system/darwin/app"       ✓ Platform-specific

# Invalid search paths
"/absolute/path"          ✗ Absolute path
"../parent/path"          ✗ Parent reference
"path with spaces"        ✗ Contains spaces
```
